# Store-System-Aldi-

A retail store system project with and Aldi theme for CSP 15100. 
This deals with object oriented programming (OOP) in which it contains:
inheritance,
abstract class,
file i/o,
hashset,
selection sort,
main method,
string class method(s),
GUI,
interface
